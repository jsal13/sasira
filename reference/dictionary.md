# Sasiran Dictionary
*The Whispered Tongue - A Language of the Ancient Ruins*

## Phonology Notes
- Emphasis on sibilants: s, z, š, ž, ṣ, ḫ
- Minimal plosives (only soft d, g rarely used)
- Vowels: a, i, u, e, o
- Other consonants: r, l, m, n, f, h

## Grammar Notes

### Word Order
Basic word order is Subject-Object-Verb (SOV), similar to Akkadian.

### Plurals
Add **-იმ** to nouns for plural:
- შალ (person) → შალიმ (people)
- აش (water) → აшიმ (waters)

### Past Tense
Add **-უმ** to verbs:
- ሳრ (to be) → ሳრუმ (was)
- სირ (to see) → სირუმ (saw)

### Future Tense
Add **-ალ** to verbs:
- ሳრ (to be) → ሳრალ (will be)
- სირ (to see) → სირალ (will see)

### Possessive
Add **-შ** for possession:
- შალ (person) → შალশ (person's)
- ሳრ (king) → ሳრშ (king's)
